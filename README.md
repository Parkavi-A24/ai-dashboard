

🧠 AI Learning Intelligence Dashboard
A modern AI-inspired full-stack dashboard to track, manage, and analyze learning progress across Artificial Intelligence topics.

This project demonstrates React + Node.js, full CRUD operations, REST APIs, and a futuristic AI-style UI.

🚀 Features
✅ Full CRUD Operations

Create AI topics
View learning progress
Update topic details
Delete topics
📊 Track multiple variables:

Topic name
Status (Not Started / In Progress / Completed)
Difficulty (Easy / Medium / Hard)
Priority (Low / Medium / High)
Notes
🤖 AI-Inspired Recommendation Banner

🎨 Futuristic dark AI-themed UI

🧩 Component-based React architecture

🔗 REST API integration

▶️ How to Run the Project Locally
1️⃣ Clone the repository
git clone https://github.com/Asifa007/ai-learning-intelligence-dashboard.git
2️⃣ Start Backend
cd backend
npm install
node server.js
3️⃣ Start Frontend
cd frontend
npm install
npm run dev
The app will run at:
http://localhost:5173
